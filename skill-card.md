# Skill Card: UGC Comment Insights Miner

| Section | Content |
|---------|---------|
| **Skill Name** | UGC Comment Insights Miner |
| **Category** | Content Enhancement / AI Automation |
| **Purpose** | Extract structured insights from community comments and discussions |
| **Input Type** | URLs or comment text (Reddit, X/Twitter, forums, blogs) |
| **Output Type** | Structured insights report (counter-arguments, FAQs, experiences, viral patterns, cross-references) |
| **Time to Output** | 1-2 minutes per request |
| **Complexity** | Medium - requires multi-step analysis and classification |
| **Use Case** | Content writers enriching blog posts with community perspectives |
| **Key Features** | URL fetching, comment classification, viral pattern analysis, blog cross-referencing |
| **Limitations** | Platform-dependent fetching, max 100 comments per URL, no CMS integration |
| **Format** | Claude Project Instructions (SKILL.md) |

## Quick Setup

1. Create new Claude Project
2. Copy contents of `SKILL.md` into Project Instructions
3. Paste URL or comments
4. Get structured insights report

## Output Sections

- Counter-Arguments & Alternative Perspectives
- Real-World FAQs
- Personal Experiences & Case Studies
- Viral Pattern Analysis
- Blog Cross-Reference (optional)
